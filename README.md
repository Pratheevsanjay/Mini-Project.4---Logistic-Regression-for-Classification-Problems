# Mini-Project.4---Logistic-Regression-for-Classification-Problems
This notebook demonstrates Logistic Regression for binary classification problems, including model training, prediction, evaluation metrics, and confusion matrix analysis.

📌 Project Overview

This project demonstrates the implementation of Logistic Regression, a supervised Machine Learning algorithm used for classification problems.

Unlike Linear Regression, Logistic Regression is used when the target variable is categorical (e.g., Yes/No, 0/1, True/False). This notebook covers data preprocessing, model training, prediction, and evaluation using classification metrics.
